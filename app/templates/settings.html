Statute Verification

Virginia Code — Verify Va. Code § X citations against the free Virginia LIS public API. No API key required.

Uses the free, unauthenticated Virginia LIS API (law.lis.virginia.gov). Results are cached to avoid redundant lookups.

Federal U.S. Code — Verify 42 U.S.C. § 1983 citations against the GovInfo API. Requires a free API key from api.data.gov. Without a key, citations remain STATUTE_DETECTED.

Leave as-is (showing ••••) to keep the current key unchanged. Get a free key at api.data.gov. When enabled and a GovInfo API key is configured, U.S.C. citations are confirmed against the United States Code and shown as STATUTE_VERIFIED. Results are cached.

Harvard Caselaw Access Project (CAP) — Secondary fallback for citations not found via CourtListener. When enabled, NOT_FOUND citations are re-queried against the CAP database. The CAP API was decommissioned in 2024; if it remains unavailable the pass is silently skipped. An optional API key permits higher-rate access.
